---
title: Raw HTML
description : "Display HTML content on a page."
---

You can use this shortcode to display HTML content on a page.

```markdown
{{< rawhtml >}} 
  <p><font size="7" color="#00ff00">This is <strong>raw HTML</strong>, inside Markdown.</font></p>
{{< /rawhtml >}}
```
